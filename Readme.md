# Tweets CLI

Follow a town's tweets with sentiment analysis

## Configuration

Add a file at `~/tweets_cli.json` that looks like this

```json
{
  "consumer_key": "",
  "consumer_secret": "",
  "token": "",
  "token_secret": ""
}
```

## Install

```
npm install bthesorceror/tweets_cli -g
```

## Usage

```
tweets_cli "Joplin, MO"
```
